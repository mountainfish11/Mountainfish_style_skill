# /mountainfish_start

开发会话开始时，分层加载 Mountainfish 记忆库。

## 使用方式

```
/mountainfish_start
```

## 执行流程

### Step 1: 检查记忆库

检查 `~/.claude/skills/mountainfish/memory/` 目录是否存在：

- **不存在**：提示用户先使用 `/mountainfish_integrate` 积累经验
- **存在**：继续执行

### Step 2: 读取分层索引

读取 `~/.claude/skills/mountainfish/memory/index.md`，获取：
- 铁律 N 条 / 指南 N 条 / 参考 N 条
- 最近更新时间

### Step 3: 分层加载记忆（v2.0）

**加载铁律（全部，~500 token）：**
读取每个 `memory/*.md` 文件中 `## 铁律` 部分的所有条目，完整注入。

**加载指南摘要（~800 token）：**
读取每个 `memory/*.md` 文件中 `## 指南` 部分，只注入：
- 条目标题
- 原则一句话
- 适用标签
- 不注入完整正文（按需时再展开）

**加载参考索引：**
仅加载参考条目的标题列表，不注入正文。

### Step 4: 设置会话标记

设置 `[Mountainfish: loaded]`，后续自动触发跳过全量加载，仅做按需检索。

### Step 5: 输出摘要

```
=== Mountainfish 记忆就绪 ===

📚 记忆库:
- 铁律: X 条（无条件生效）
- 指南: X 条（按需加载全文）
- 参考: X 条（标签匹配检索）

🔴 核心铁律：
1. [铁律 1 规则一句话]
2. [铁律 2 规则一句话]
3. [铁律 3 规则一句话]

📅 最近更新：YYYY-MM-DD

上下文预算: ~1300 token（铁律 500 + 指南摘要 800）
✅ 记忆已就绪
```

### Step 6: 分层注入

将铁律完整内容 + 指南摘要 + 参考索引注入到当前会话上下文：

```
=== Mountainfish 注入内容 ===
## 铁律（以下规则无条件遵循）
[铁律 1 完整内容]
[铁律 2 完整内容]
...

## 指南摘要（需要时可展开全文）
- [指南 1 标题]: [原则一句话]  `#标签`
- [指南 2 标题]: [原则一句话]  `#标签`
...

## 参考索引（按需检索）
- [参考 1 标题]  `#标签`
- [参考 2 标题]  `#标签`
...
```
